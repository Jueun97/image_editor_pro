## 프로젝트 소개
그림그리기, 배경색 추가하기, 텍스트 추가하기 등 간단하게 이미지를 꾸밀 수 있는 프로그램

### 사용 기술
* HTML (Canvas)
* CSS
* Javascript

### 주요 기능
#### 1. 이미지 업로드
#### 2. 드로잉 작업
- 색 변경 가능
- 브러쉬 크기 변경 가능
#### 3. 배경색 입히기
- 색 변경 가능
- 그라데이션 효과 (색, 방향 조절 가능)
- 투명도 조절 가능 
#### 4. 텍스트 추가하기
